# project-Escape,Meow
# 플레이어는 스탯을 관리하며 집사의 루틴을 피해 탈출을 준비하는 3D 고양이 탈출 게임.
🐾 게임 목표
집사의 사랑을 피해
최소 일수로 탈출하라

✔ 3인칭 카메라
✔ 무표정 고양이
✔ 행동 기반 RPG 스탯
✔ 패턴형 집사 루틴
✔ 시스템 자막 개그
✔ 탈출하면 즉시 종료
✔ 최소 일수 랭킹

unreal engine 5 와 Blender 이용
3인칭 뷰 
랭킹은 몇일차에 탈출 성공했는가로 갈림
주인공 하루 행동 포인트 5개로 제한, 모두 사용시 다음날로 넘어감
고양이는 말 못하고 야옹 냐옹 등으로 대체
필요한 설명은 시스템 자막으로 설정
표지는 좋아죽겠는 집사한테 안겨서 삐질거리는 고양이!
